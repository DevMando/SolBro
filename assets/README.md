# Assets

Drop your README images in here using these filenames so the existing references resolve:

| Filename | Where it appears | What to capture |
|---|---|---|
| `hero-conversation.png` | Just below the intro paragraph | A real Discord conversation showing SolBro replying in character. At least one back-and-forth turn so the personality reads. ~1200px wide. |
| `tool-use.png` (or `.gif`) | Top of the Feature tour section | SolBro actively using a tool — vision describing a dropped image, weather lookup, GIF drop, or video re-upload. **A GIF is ideal here** — animation captures liveness. |
| `personality-variations.png` | Top of the "Defining your bot's personality" section | 2–3 mini conversations showing the same bot with different system prompts (default vs pirate vs code reviewer, etc.). Collage or stacked screenshots both work. |

## Tips

- **PNG for static, GIF for animation.** JPG also fine for screenshots, but PNG keeps text sharp.
- **Aim for ~1200px wide.** GitHub renders the README at varying widths; anything narrower looks pixelated on retina displays.
- **Crop tight.** Cut out empty channel sidebars and unrelated chatter; readers should see the bot's reply within the first second of looking.
- **Privacy:** blur or rename any usernames you don't want public before exporting.

If you want to add more images later (architecture diagram, setup screenshots, etc.), drop them in this folder and reference them from wherever in the README makes sense.
